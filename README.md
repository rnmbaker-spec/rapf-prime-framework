# rapf-prime-framework

Recursive Admissibility Prime Framework (RAPF) — Paper 1 (Discrete Geometry) and Paper 2 (Equidistribution Census).

The `papers/` subdirectories contain the current, corrected source files and compiled PDFs:
- **Paper 1**: `papers/1-discrete-geometry/` — The discrete information-geometric framework for twin primes
- **Paper 2**: `papers/2-equidistribution/` — Equidistribution census across admissible residue classes (n=3 through n=8)

The companion **Paper 3** (Prime Triplets + mod-30/mod-60 lattices) lives in [rapf-prime-triplets](https://github.com/rnmbaker-spec/rapf-prime-triplets).
